# 阶段二：设计系统与前端框架

> 所属版本：V3.0 | 预估工期：1 周 | 优先级：P0
> 前置条件：阶段一完成（项目骨架可运行）
> 产出物：可用的光影视觉设计系统、主题切换、动画系统、布局组件、公共样式层

---

## 一、目标

将光影设计语言（Light & Shadow Design Language）落地为可用的前端代码——CSS 变量、Tailwind 配置、SCSS 公共样式层、GSAP 动画系统、布局组件，确保后续页面开发能直接引用设计 Token 而非硬编码颜色/间距/光影。

### 成功标准

- [ ] 暗色/亮色主题一键切换，无闪烁（FOUC）
- [ ] 所有设计 Token 通过 CSS 变量访问，不使用硬编码色值
- [ ] 光影效果（发光、边缘高光、渐变背景）在各模式下正确渲染
- [ ] 动画系统封装为 Vue 组件/指令，支持 `prefers-reduced-motion`
- [ ] 布局组件（AppHeader、AppFooter、AppLayout）可复用
- [ ] 公共样式层结构清晰，组件样式不碎片化
- [ ] Lighthouse 性能评分 > 90（空页面状态）

### 2.4 字体资产与排版策略

本项目共引入 6 款字体，经 `fontTools` 技术解析后，按角色分为五大排版层级：

| 角色 | 选用字体 | 原始体积 | 子集化后 | 用途 |
|------|---------|---------|---------|------|
| **正文阅读** | 思源黑体 SC | 16 MB | **1.2 MB** | 博客正文、段落、说明文字 |
| **代码展示** | JetBrains Mono Bold | 135 KB | **135 KB** | 代码块、终端模拟（西文） |
| **代码展示** | 更纱黑体 Mono SC | 13 MB | **20 KB** | 代码块中文回退（等宽对齐） |
| **英文展示标题** | LIBRARY 3 AM | 34 KB | **34 KB** | Hero 英文大标题、装饰性标语（仅含字母，无数字/标点） |
| **中文特色标题** | 字言诗动漫二次元 | 5 MB | **174 KB** | 特色区块标题、引用装饰 |
| **像素/复古风** | 苹苹果果像素体 | 8 MB | **191 KB** | 游戏/复古项目 Demo 的标题 |

#### 排版场景分配

| 场景 | 字体变量 | 说明 |
|------|---------|------|
| Hero 英文主标题 | `font-display` | LIBRARY 3 AM 的艺术装饰感营造视觉锚点（仅限纯英文单词，避免数字/标点） |
| Hero 中文副标题 | `font-accent` | 动漫圆体的活泼感与英文标题形成对比 |

### 2.5 图标系统设计

图标严格遵循 **光影设计语言**，采用 24×24 线框风格，通过 CSS `filter: drop-shadow()` 实现交互光晕。

#### 设计规范

| 规范项 | 设定 |
|--------|------|
| 画布 | 24×24 viewBox |
| 描边 | 2px（线框清晰有力，契合玻璃质感与光影层次） |
| 线端 | 全圆角（round cap / round join） |
| 填充 | 无填充，纯线框 |
| 颜色 | `currentColor`（跟随 CSS 颜色变量，明暗主题自适应） |
| 无障碍 | 每个图标带 `aria-label` 中文标签 |

#### 光影交互特性

```css
/* hover 时产生主色光晕 */
.icon-glow:hover {
  filter: drop-shadow(0 0 6px rgba(37, 99, 235, 0.4));
  transition: filter 0.3s ease;
}

/* 当前激活导航项使用渐变描边 */
.icon-active {
  stroke: url(#icon-gradient);
}
```

`sprite.svg` 中预置共享滤镜：
- `filter="url(#icon-glow)"` —— 图标发光效果
- `url(#icon-gradient)` —— 渐变描边（用于强调状态）

## 三、公共样式管理规范

前端样式采用 **"公共样式层 + 组件级覆盖"** 双层架构，所有全局 Token、动画、混合宏统一收拢至 `styles/` 目录。

### 3.1 目录结构

```
styles/
├── _variables.scss        # 设计 Token（色彩、间距、阴影、字体、光影）
├── _lightTheme.scss       # 亮色模式变量覆写
├── _darkTheme.scss        # 暗色模式变量覆写
├── _mixins.scss           # SCSS 混合宏（玻璃拟态、光影效果、响应式断点）
├── _animations.scss       # 全局动画关键帧与工具类
├── _globals.scss          # 全局基础样式（html, body, 滚动条, 文本选中色）
├── _utilities.scss        # 原子化工具类（超出 Tailwind 范围的自定义类）
└── main.scss              # 入口文件：按序导入以上所有模块
```

### 3.2 规范原则

1. **禁止硬编码色值**：所有颜色必须通过 CSS 变量引用（`var(--color-primary)`），禁止在组件中写死 `#2563EB` 等色值。
2. **禁止组件内重复定义通用动画**：进入动画统一使用 `_animations.scss` 中的工具类。
3. **组件级样式仅写布局与特殊覆盖**：组件 `.vue` 文件中的 `<style scoped>` 只负责该组件特有的定位、宽高、特殊交互状态，不涉及颜色、字体、阴影等全局 Token。
4. **光影效果统一由 `_mixins.scss` 提供**：确保全站发光强度、模糊半径、过渡时长保持一致。

### 3.3 入口文件示例

```scss
// styles/main.scss
@use "variables" as *;
@use "lightTheme" as *;
@use "darkTheme" as *;
@use "mixins" as *;
@use "animations" as *;
@use "globals" as *;
@use "utilities" as *;
```

在保留玻璃拟态通透感的基础上，引入边缘高光与柔和光晕，营造层次丰富的光影质感。

### 4.2 带光影的卡片组件类

```scss

    -webkit-mask-composite: xor;

    pointer-events: none;
  }

  &:hover {
    box-shadow: var(--shadow-lg), var(--glow-primary);

  }
}

.dark .card-glass {
  @include glass-dark;

  &:hover {
    box-shadow: var(--shadow-lg), var(--glow-primary);
  }

// 按钮光影样式
.btn-glow {

  box-shadow: var(--glow-inner-light), var(--glow-soft);

  &:hover {
    box-shadow: var(--glow-inner-light), var(--glow-primary);
```

### 4.3 导航栏光影

```scss
// styles/_utilities.scss

.glass-nav {
  @include glass-nav;

  // 滚动后加深阴影与微弱光晕
  &.scrolled {
    box-shadow: var(--shadow-md);
  }
}

.dark .glass-nav {
  @include glass-nav-dark;
}
```

> **性能约束**：同时生效的 `box-shadow` / `filter: drop-shadow()` 元素不得超过 10 个。复杂发光效果优先使用伪元素 `::before` 实现，减少主层合成开销。

## 五、动画系统

### 5.1 性能预算

| 指标 | 上限 | 测量方式 |
|------|------|---------|
| 首页动画总时长 | < 2.0s | Chrome DevTools Performance |
| 单次滚动触发动画 | < 0.3s | `scrollTrigger` duration |
| 动画帧率 | > 50fps | FPS Meter |
| `will-change` 同时生效元素 | < 5 个 | 手动审查 |

### 5.2 CSS 弹性动画（纯 CSS，不依赖 GSAP）

```scss
// styles/_animations.scss

// 弹性向上淡入
@keyframes fadeInUpElastic {
  0%   { opacity: 0; transform: translateY(25px); }
  70%  { opacity: 1; transform: translateY(-3px); }
  100% { opacity: 1; transform: translateY(0); }
}

// 弹性从左侧滑入
@keyframes slideInLeftElastic {
  0%   { opacity: 0; transform: translateX(-30px); }
  70%  { opacity: 1; transform: translateX(5px); }
  100% { opacity: 1; transform: translateX(0); }
}

// 弹性缩放进入
@keyframes scaleInElastic {
  0%   { opacity: 0; transform: scale(0.85); }
  70%  { opacity: 1; transform: scale(1.03); }
  100% { opacity: 1; transform: scale(1); }
}

// 光晕脉冲（用于时间线节点、强调元素）
@keyframes glowPulse {
  0%, 100% { box-shadow: 0 0 8px rgba(37, 99, 235, 0.3); }
  50%      { box-shadow: 0 0 20px rgba(37, 99, 235, 0.5); }
}

.animate-fade-in-up {
  animation: fadeInUpElastic 0.6s ease forwards;
```

## 六、主题切换系统

### 6.1 Composable

```typescript
// composables/useTheme.ts
// 主题切换逻辑：优先 localStorage，其次系统偏好
export const useTheme = () => {
  const colorMode = useColorMode(); // Nuxt 内置

  const isDark = computed(() => colorMode.value === 'dark');

  const toggleTheme = () => {
    colorMode.preference = isDark.value ? 'light' : 'dark';
  };

  // 初始化：优先从 localStorage 读取，其次系统偏好
  onMounted(() => {
    const stored = localStorage.getItem('nuxt-color-mode');
    if (!stored) {
      const prefersDark = window.matchMedia('(prefers-color-scheme: dark)').matches;
      colorMode.preference = prefersDark ? 'dark' : 'light';
    }
  });

  return { isDark, toggleTheme };
};
```

### 6.2 防闪烁策略（关键）

```typescript
// nuxt.config.ts
export default defineNuxtConfig({
  colorMode: {
    preference: 'system',
    fallback: 'light',
    classSuffix: '',     // 用 .dark class，不用 .dark-mode
    storageKey: 'theme', // localStorage key
  },
  // 内联脚本：在 HTML 解析前设置 .dark class，避免 FOUC
  app: {
    head: {
      script: [
        {
          innerHTML: `
            (function() {
              const theme = localStorage.getItem('theme');
              const prefersDark = window.matchMedia('(prefers-color-scheme: dark)').matches;
              if (theme === 'dark' || (!theme && prefersDark)) {
                document.documentElement.classList.add('dark');
              }
            })();
          `,
          type: 'text/javascript',
        },
      ],
    },
  },
});
```
